** Copy configuration files
The next script copy the config file to its designed location
#+name: copy_configs_files
#+begin_src sh
  cp .emacs ~/
  cp .tmux.conf ~/
  cp .gitconfig ~/
  cp .zshrc ~/
  cp home.nix ~/.config/nixpkgs/
#+end_src

** Dependencies
- oh-my-zsh
- Tmux manager
